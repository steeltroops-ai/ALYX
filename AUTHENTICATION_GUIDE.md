# ALYX Authentication Guide

## How to Access the System

### Option 1: Use Demo Accounts (Immediate Access)

You can immediately log in using these pre-configured demo accounts:

#### **Admin Account** (Full System Access)
- **Email**: `admin@alyx.physics.org`
- **Password**: `admin123`
- **Permissions**: Full system access, user management, system configuration

#### **Physicist Account** (Analysis & Job Submission)
- **Email**: `physicist@alyx.physics.org`
- **Password**: `physicist123`
- **Permissions**: Submit jobs, access data, create notebooks, advanced visualization

#### **Analyst Account** (Read-Only Analysis)
- **Email**: `analyst@alyx.physics.org`
- **Password**: `analyst123`
- **Permissions**: View results, basic visualization, read data access

### Option 2: Create New Account (Registration)

1. **Go to the Login Page**: Navigate to `/login` in your browser
2. **Click "Create one here"**: Look for the registration link at the bottom of the login form
3. **Fill out the Registration Form**:
   - **Email**: Your institutional email address
   - **First & Last Name**: Your full name
   - **Organization**: Select from dropdown (CERN, FERMILAB, DESY, etc.)
   - **Role**: Choose between:
     - **Analyst**: Read-only access to analysis results
     - **Physicist**: Full analysis access and job submission
   - **Password**: Minimum 8 characters
